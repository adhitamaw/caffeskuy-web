# CoffeSkuy

A modern web application for cafe review and discovery built with Laravel. Discover the best cafes, share your experiences, and connect with fellow coffee enthusiasts. 

CoffeSkuy provides a comprehensive platform where coffee lovers can explore new cafes, read authentic reviews from the community, and share their own cafe experiences. Whether you're looking for the perfect workspace cafe, a cozy spot for a date, or simply want to discover hidden gems in your area, CoffeSkuy makes it easy to find exactly what you're looking for.

Our platform combines the power of user-generated content with advanced search capabilities, allowing you to filter cafes by location, rating, price range, and specific amenities. With features like personal favorite lists, detailed photo galleries, and real-time reviews, CoffeSkuy helps you make informed decisions about where to spend your precious coffee time.

Join thousands of coffee enthusiasts who trust CoffeSkuy to guide them to their next great cafe experience.

## Features

- **Cafe Discovery** - Search and explore cafes by location, rating, and category
- **Review System** - Rate cafes and share detailed reviews with the community  
- **Personal Favorites** - Save your favorite cafes for quick access
- **User Dashboard** - Manage your profile, reviews, and favorite locations
- **Admin Panel** - Comprehensive management system for cafes and user content
- **Secure Payments** - Integrated Stripe payment system for premium features

## Technologies

- **Laravel 11** - Modern PHP framework for robust backend development
- **MySQL** - Reliable database management system
- **Bootstrap** - Responsive frontend framework for beautiful UI
- **Stripe** - Secure payment processing integration
- **Laravel Sanctum** - API authentication system
- **SweetAlert2** - Enhanced user notifications and alerts
